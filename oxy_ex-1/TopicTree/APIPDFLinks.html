API PDF Links

The Blume Global API documentation site caters to users the instructions on how to effectively use and integrate the Blume Global API. It also provides updates on the API's lifecycle such as new versions or retirement. Some aspects of API documentation can be generated automatically via Swagger or other documents.

This section contains links to all Blume API Documentation PDFs.

Table 1. Visibility API Documentation Links
Document Title Description
Blume Visibility Flight Status API Integration Guide This guide provides instructions for integrating the Blume Visibility Flight Status API, which allows users to track the status of flights in real time.
Blume Visibility Geo Location API Integration Guide (Pull) This document explains how to integrate the Blume Visibility Geo Location API using a pull mechanism to retrieve geographical location data for tracking purposes.
Blume Visibility Get Event API Integration Guide (Push) This guide describes the process for integrating the Blume Visibility Get Event API using a push mechanism to receive real-time event notifications.
Blume Visibility Milestones API Integration Guide (Pull) This document provides instructions for integrating the Blume Visibility Milestones API using a pull method to retrieve milestone information for shipments.
Blume Visibility Milestones API Integration Guide (Push) This guide explains how to integrate the Blume Visibility Milestones API using a push method to receive real-time updates on shipment milestones.
